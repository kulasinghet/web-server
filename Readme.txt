1. To start the web server please run the corresponding bat/sh file.
	- if you are in a windows environment please run the windows.bat 
	- if you are in a linux/mac environment please run the mac.sh

	if you like to proceed manualy, then open WebServer folder in the terminal, and change directory to out\production\WebServer. From there you should find Server.class java class file. Then type and run "java WebServer" 

	>> Terminal code to start the web server <<
		
		cd out\production\WebServer
		java Server



2. Then open your web browser and try to access http://localhost:2728/

From there you should see the default html file.
